# Podman

Podman: A tool for managing OCI containers and pods.

## Prerequisites

- Kubernetes(depends on the app requirements)
- sealos 4.x.x

## Installing the app

To install the app with sealos run  command:

```bash
sealos run docker.io/labring/podman:v4.5.1
```

These commands deploy podman binary on the Kubernetes cluster in the `/usr/local/bin` directoryï¼Œlist app using

```bash
$ which podman
```
